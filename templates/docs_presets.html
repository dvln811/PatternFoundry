Instrument Presets Reference

This page documents every setting for each of the 14 built-in instrument presets. Use it as a reference when creating custom instruments in the Chart Designer, or to understand why each instrument behaves the way it does.

ESNQCLRTY MESMNQMCLM2K SPYAAPLMSFTPLTR TSLAGME
How to read this page

Each instrument has two tables: Historical (candle generation - regime, drift, volatility, wicks) and Microstructure (tick engine - LOB, agents, Hawkes). The "Effect of Change" column tells you what happens if you increase or decrease that setting in the Chart Designer.

MES - Micro E-mini S&P

Identical price action to ES. 1/10th the dollar value per tick ($1.25 vs $12.50). Same historical settings as ES. Designed for smaller accounts.

Note

MES, MNQ, MCL, and M2K use the exact same regime, drift, volatility, and wick settings as their E-mini counterparts. Only tick_value and margin differ.

MNQ - Micro Nasdaq

Identical price action to NQ. $0.50/tick vs $5.00. Same historical and microstructure settings as NQ.

Identical price action to CL. $1.00/tick vs $10.00. Same historical settings as CL.

M2K - Micro Russell 2000

Identical price action to RTY. $0.50/tick vs $5.00. Same historical settings as RTY.

Tuning Guide

When creating custom instruments in the Chart Designer, use these guidelines:

Want more trending sessions?

  • Increase trend duration (40+ bars for strong trending)
  • Decrease trend → chop transition (below 0.30)
  • Increase trend_magnitude (larger drift per bar)
  • Add impulse → trend transitions (spikes lead into trends)

Want more choppy/range-bound sessions?

  • Increase chop duration (30+ bars)
  • Increase trend → chop transition (0.60+)
  • Decrease trend_magnitude
  • Increase wick chop_ratio (more rejection candles)

Want more volatile/impulsive behavior?

  • Increase all volatility values proportionally
  • Increase impulse_magnitude
  • Increase chop → impulse and trend → impulse transitions
  • Increase hawkes_alpha in microstructure (more tick clustering)

Want wider spreads / worse fills?

  • Increase spread_base (2-4 ticks for illiquid instruments)
  • Increase spread_vol_mult (3-5 for thin books)
  • Decrease lob_base_size (less depth = more slippage)

Want more realistic S/R interaction?

  • Increase pool_strength (0.30-0.40 for sticky levels)
  • Increase pool_count (4-5 for instruments with many key levels)
  • Increase mean_rev_strength (for VWAP-dominated instruments)
